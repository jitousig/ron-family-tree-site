# Ron's Family Tree — published site

The interactive Tourangeau–Casey family tree, published via GitHub Pages:

**https://jitousig.github.io/ron-family-tree-site/**

`index.html` is a single, self-contained build (portraits embedded; Wikipedia links/previews need
internet). It is generated from the private source repository **jitousig/ron-family-tree** — do not edit
`index.html` here by hand; regenerate it there and copy the new build over.

Living family members are kept private/summarized. Records and research notes live only in the private
source repo.
